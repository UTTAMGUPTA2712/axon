Privacy Policy

Last Updated: January 12, 2026

Ask AI - SCreen Assistant ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how our Chrome extension processes your information.

1. Information Collection and Use

  • User Input & Screen Data: When you trigger the extension, we capture the selected text or a screenshot of your active tab.
  • Personalization Data: We may store user preferences, facts, and professional details (your "Digital Twin") derived from your interactions to improve response relevance.
  • Processing: This data is sent securely to Groq API for processing.
  • No External Storage: We do not store, log, or share your screen data, text selections, profile data, or API keys on our servers. All data is stored locally in your browser (chrome.storage.sync) or processed ephemerally.

2. Third-Party Services

We use Groq Cloud as our primary AI inference provider. Please refer to Groq's Privacy Policy for details on how they handle API request data.

3. API Keys

The extension allows you to bring your own API Key or use a default configuration. In both cases, the key is stored locally on your device using Chrome's secure storage API (chrome.storage.sync). It is never transmitted to us or any third party other than the Groq API for authentication.

4. Changes to This Policy

We may update this policy from time to time. Any changes will be posted on this page.

5. Contact

If you have questions about this policy, please contact us via the Chrome Web Store support page or our GitHub repository.
